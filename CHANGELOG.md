# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-03-17

### Added
- Plan-then-execute workflow with safety confirmation codes
- Multi-project support with session persistence
- Telegram-based remote control of Claude Code
- PDF response delivery for plans and execution results
- Research mode for read-only analysis
- Transcript search across media library (optional)
- Auto re-authentication for expired Claude sessions
- Permission management per project (`!allow` command)
- NATO phonetic safety codes (e.g., `foxtrot-4097`)
- Environment-based configuration with `.env` support
